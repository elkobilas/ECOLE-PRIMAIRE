# 🚀 Guide de Démarrage - Application de Gestion Scolaire

## ⚠️ Problème identifié
L'application ne peut pas être lancée directement en ouvrant `index.html` dans le navigateur car elle utilise des modules JavaScript et des API qui nécessitent un serveur HTTP.

## ✅ Solutions disponibles
## 📱 Accès à l'application

Une fois le serveur démarré :
- **URL principale :** http://localhost:8080
- **Page de connexion :** http://localhost:8080/login.html
- **Page de profil :** http://localhost:8080/profile.html
- **Test :** http://localhost:8080/test-simple.html

## 🔐 Comptes de test

L'application inclut des comptes de démonstration :
- **Admin :** `admin` / `admin123`
- **Enseignant :** `enseignant` / `enseignant123`
- **Direction :** `direction` / `direction123`

## 📞 Support

Si vous rencontrez encore des problèmes :
1. Vérifiez que tous les fichiers sont présents
2. Testez avec `test-simple.html` d'abord
3. Consultez la console du navigateur (F12)
4. Vérifiez que le port 8080 n'est pas utilisé par une autre application
